# UX Design Briefs

This folder contains UX Design Briefs produced by the **UI/UX Designer** agent for each feature.

## Current Briefs

- [dynamic-ci-smoke-tests-ux-brief.md](dynamic-ci-smoke-tests-ux-brief.md) - Developer/operator UX for CI smoke result readability and failure triage flow.

## Naming Convention

```
<feature-slug>-ux-brief.md
```

## Brief Structure

Each brief includes:
- **User journey map** — entry points, happy path, error paths, edge cases
- **Interaction specifications** — per-action behavior, state transitions, timing
- **Feedback & states** — loading, empty, error, success, disabled states
- **Accessibility checklist** — ARIA roles, keyboard nav, focus, contrast, screen reader
- **Responsive considerations** — mobile-first breakpoint behavior
- **Open questions resolved** — questions asked and answers received
- **Revision notes** (if revising) — before/after description of UX changes
